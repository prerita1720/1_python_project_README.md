## 1️⃣ Personal Data Collector

Collects a user's personal details (name, age, height, weight, email, birthdate), casts inputs to the correct types, inspects them with `type()` and `id()`, and classifies the user into an age group (Child / Teenager / Young Adult / Senior Citizen) using `if / elif / else`.

**Concepts:** User input, type casting, data types & memory identity, conditional logic

---

## 2️⃣ Pattern Generator & Number Analyzer

A menu-driven program offering star-pattern generation or number-range analysis (odd/even classification plus running sum) using a `for...else` block.

**Concepts:** `while` loops, `for` + `range()`, `for...else`, modulus operator, f-strings

---

## 3️⃣ Student Data Organizer

A CRUD-style console app for managing student records — add, view, update, delete, and track subjects offered — using a list of dictionaries, with a set for unique subjects and a tuple for ID/DOB pairing.

**Concepts:** Functions, lists, dictionaries, sets, tuples, `for...else` for search

---

## 4️⃣ Data Analyzer & Transformer

Lets the user input a list of numbers and run summary statistics, filtering, sorting, and factorial calculations, all through a function-per-feature menu design.

**Concepts:** Functions, `global` variables, list comprehensions, `sum`/`min`/`max`/`sort`

---

## 5️⃣ Employee Management System (OOP)

Models a `Person → Employee → Developer` inheritance chain, with each subclass overriding `display()` to show progressively more detail — a clean demonstration of inheritance and polymorphism.

**Concepts:** Classes, `super()`, method overriding, polymorphism

---

## 6️⃣ Personal Journal Manager

A timestamped journal app that adds, views, searches, and deletes entries stored in a text file, using targeted exception handling for missing files and permission errors.

**Concepts:** File modes (`x`, `a`, `r`), exception handling, classes, `datetime`

---

## 7️⃣ Multi-Utility Toolkit

A nested-menu toolkit combining datetime tools (stopwatch, countdown, date diff), math operations (factorial, compound interest, trig, circle area), random generators (numbers, passwords, OTPs), file operations, and a runtime module explorer using `dir()`.

**Concepts:** Nested menus, `datetime`, `time`, `math`, `random`, `uuid`, dynamic imports

---

## 8️⃣ Smart Expense Tracker

An expense-tracking app backed by a CSV file — auto-creates itself with sample data, validates and appends new expenses, and visualizes spending with a 4-panel dashboard (bar, line, pie, histogram).

**Concepts:** `pandas` CSV persistence, data cleaning, `pd.concat`, `groupby`, `matplotlib`/`seaborn` multi-chart dashboard

---

## 9️⃣ Sales Data Analyzer

A full analysis pipeline class covering data loading, exploration, cleaning, derived columns, sorting/filtering, aggregation, statistics, pivot tables, and seven different chart types (bar, line, scatter, histogram, stacked area, boxplot, heatmap).

**Concepts:** OOP data pipeline, `groupby`, `pivot_table`, `.describe()`/`.std()`/`.var()`, wide range of `matplotlib`/`seaborn` visuals

---

## 🔟 Student Performance Analysis

Builds a student dataset from scratch, calculates total/percentage scores, compares performance by gender and subject, and visualizes results with bar charts and a histogram.

**Concepts:** DataFrame from a dictionary, derived columns, `groupby`, bar chart & histogram visualization

---

## 1️⃣1️⃣ Indian Retail Business Sales Analysis

A business-focused EDA on a large retail sales dataset — cleaning data, analyzing revenue by region/category/customer type, deriving monthly sales trends, correlating discount with revenue, building a 6-visual dashboard, and closing with concrete business recommendations.

**Concepts:** Real-world data cleaning, multi-dimension `groupby`, time-based aggregation, correlation analysis, dashboard visualization, business insight generation
